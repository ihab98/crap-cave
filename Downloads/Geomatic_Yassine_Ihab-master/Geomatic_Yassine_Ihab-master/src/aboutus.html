ApolloGeo

ApolloGeo is a software suite that can be used to extract GIS data from Imajbox mobile mapping systems. ApolloGeo offers a way to view, edit and process GIS data. The suite offers tools to synchronize image data with map content, extract 3D coordinates from images, in addition to object detection and classification for transportation and utilities inventory generation.

ApolloGeo Web Service

ApolloGeo is also web service that provides access to already collected GIS data nationwide.
